# 多维表格插件 - 数据导出

这是一个基于飞书多维表格（Bitable）开发的边栏插件，帮助用户将数据表中的数据导出为 CSV 或 JSON 格式文件。

## 核心功能

-   **数据导出**：将多维表格数据导出为 CSV（Excel兼容）或 JSON 格式
-   **字段选择**：支持选择需要导出的字段列，按需导出
-   **顺序一致**：导出数据的字段顺序和行顺序与当前视图显示完全一致
-   **格式兼容**：支持多种字段类型（文本、数字、单选、多选、日期、附件、用户等）

## 重要说明

> **只要能看到数据，就可以下载！**
> 
> 即使表格设置了"禁止复制"权限，只要您能查看数据表内容，就可以使用本插件将数据导出下载。

## 使用方法

1. 打开飞书多维表格，在右侧边栏找到"数据导出"插件
2. 选择导出格式：
   - **CSV**：适合用 Excel 打开查看和编辑
   - **JSON**：适合程序处理和数据交换
3. 勾选需要导出的字段（默认全选）
4. 点击"导出"按钮，文件将自动下载到本地

## 开发与部署

### 环境要求
-   Node.js 16.19.0+
-   npm

### 安装依赖
```bash
npm install
```

### 本地开发
```bash
npm run dev
```

### 打包构建
```bash
npm run build
```

-   React + TypeScript
-   Vite
-   @lark-base-open/js-sdk
